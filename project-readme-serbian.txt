# Систем за Управљање Учењем (LMS)

## Преглед

Овај Систем за Управљање Учењем (LMS) је веб апликација дизајнирана да олакша онлајн образовање. Обезбеђује платформу наставницима за управљање часовима, креирање предавања и задатака, а ученицима за приступ материјалима курса и предају задатака.

### Кључне функције:

- Аутентификација корисника (наставници и ученици)
- Управљање часовима
- Креирање и управљање предавањима
- Креирање и управљање задацима
- Контрола приступа заснована на улогама

## Технолошки стек

- Frontend: React.js са Material-UI
- Backend: Node.js са Express.js
- База података: PostgreSQL
- API упити: React Query
- Рутирање: React Router
- Аутентификација: JSON Web Tokens (JWT)

## Предуслови

Пре него што почнете, уверите се да имате инсталирано следеће:

- Node.js (v14 или новији)
- npm (обично долази са Node.js)
- PostgreSQL (v12 или новији)
- Git

## Упутства за подешавање

1. Клонирајте репозиторијум:
   ```
   git clone https://github.com/your-username/lms-project.git
   cd lms-project
   ```

2. Подесите backend:
   ```
   cd backend
   npm install
   ```

3. Подесите базу података:
   - Креирајте PostgreSQL базу података за пројекат
   - Ажурирајте конфигурацију базе података у `backend/knexfile.js` са вашим креденцијалима за базу података

4. Покрените миграције базе података:
   ```
   npx knex migrate:latest
   ```

5. Подесите frontend:
   ```
   cd ../frontend
   npm install
   ```

6. Креирајте `.env` фајл у backend директоријуму са следећим садржајем:
   ```
   PORT=5000
   JWT_SECRET=your_jwt_secret_here
   DATABASE_URL=your_postgres_connection_string_here
   ```
   Замените `your_jwt_secret_here` са сигурним случајним стрингом и `your_postgres_connection_string_here` са вашим стварним стрингом за конекцију са базом података.

## Покретање апликације

1. Покрените backend сервер:
   ```
   cd backend
   npm start
   ```
   Сервер би требало да почне да ради на `http://localhost:5000`

2. У новом терминалу, покрените frontend развојни сервер:
   ```
   cd frontend
   npm start
   ```
   React апликација би требало да се отвори у вашем подразумеваном прегледачу на `http://localhost:3000`

## Употреба

- Идите на `http://localhost:3000` у вашем веб прегледачу
- Пријавите се као наставник или ученик (можда ћете морати да убаците неке почетне корисничке податке)
- Наставници могу да креирају и управљају часовима, предавањима и задацима
- Ученици могу да виде своје додељене часове, приступају предавањима и виде задатке
